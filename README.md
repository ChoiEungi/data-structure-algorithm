## ArrayList

>  A data structure consisting of a collection of elements (values or variables), each identified by at least one array index or key. 



### Main method

- space: O(n)
- at(i) : A[i] , return index without removing, O(1)
- set(i, o) : A[i] = o , O(1)
- insert : worst : O(n), 삽입될 때 size를 늘림
- erase  : worst O(n)
- size : O(1)
- empty O(1)
